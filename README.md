# 'Omae wa' Command Line Sound

> Omae wa mou shindeiru! Nani!??

Play famous "Omae wa" sound.

## Compatibility

- Linux
- Windows
- Mac

## Installing
Install globally:

    npm install -g gemidao1

## Running
Run command:

    gemidao1

Or import in NodeJS script:

    import gemidao1 from 'gemidao1';

    gemidao1();
